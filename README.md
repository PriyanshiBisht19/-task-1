# Data Cleaning - Netflix Dataset

## Steps Performed:
1. Checked dataset shape and columns
2. Identified missing values
3. Filled missing rating and date_added values
4. Removed duplicate rows
5. Converted date_added to datetime format
6. Standardized column names
7. Fixed data types
8. Saved cleaned dataset

## Tools Used:
- Python
- Pandas

## Output:
Cleaned dataset ready for analysis.
